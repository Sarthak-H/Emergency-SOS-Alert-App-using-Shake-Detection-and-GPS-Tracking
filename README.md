# Emergency-SOS-Alert-App-using-Shake-Detection-and-GPS-Tracking

## 📌 Features
- 📳 Shake Detection using Accelerometer
- 📍 Real-time GPS Location Tracking
- 📩 Automatic SMS Alert to Emergency Contacts
- 👤 Predefined Contact Management
- ⚡ Fast and Reliable SOS Trigger

## 🛠️ Technologies Used
- Java
- Android Studio
- Google Fused Location Provider API
- SMS Manager

## ⚙️ How It Works
1. User shakes the mobile device
2. App detects motion using accelerometer
3. Fetches current location using GPS
4. Sends SOS message with location link via SMS

## 🚀 Future Enhancements
- Live location sharing
- Push notifications
- Emergency siren
- Background service support

## 📌 Conclusion
This project enhances personal safety by providing a quick and automated way to send emergency alerts with location details.
